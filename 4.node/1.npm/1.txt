node package manager : npm은 js 모듈 설치하는 툴
    패키지는 모듈의 총합
        모듈은 js 앱이 사용하는 프레임워크, 라이브러리 총칭
            프레임워크: vue, angular
            라이브러리: react // jexcel, gojs, eslint ...

    npm i fs
        node_modules : 의존성(dependancies) 모듈들이 들어간 폴더
        package.json : 의존성 모듈 리스트
        package-lock.json : 의존성들의 버전을 잠금
